# 04 — AI Tools Overview

A reference guide to the most commonly used AI tools, their strengths, and when to use them.

---

## Conversational AI Assistants

### Claude (Anthropic)
- **Best for:** Long documents, nuanced reasoning, writing, analysis, coding
- **Strengths:** Large context window, careful responses, strong at following complex instructions
- **Access:** [claude.ai](https://claude.ai) | API via Anthropic
- **Models:** Claude Opus (most capable), Claude Sonnet (balanced), Claude Haiku (fast)

### ChatGPT (OpenAI)
- **Best for:** General-purpose tasks, image input, plugin/tool ecosystem
- **Strengths:** Wide adoption, GPT-4o with vision, code interpreter, web browsing
- **Access:** [chat.openai.com](https://chat.openai.com) | API via OpenAI
- **Models:** GPT-4o, GPT-4o mini, o1 (reasoning)

### Gemini (Google)
- **Best for:** Google Workspace integration, multimodal tasks
- **Strengths:** Native integration with Gmail, Docs, Sheets; strong at web-connected queries
- **Access:** [gemini.google.com](https://gemini.google.com) | Google Workspace add-ons
- **Models:** Gemini 1.5 Pro, Gemini Flash

### Microsoft Copilot
- **Best for:** Microsoft 365 users — Word, Excel, Teams, Outlook
- **Strengths:** Deep Office integration, meeting summaries, document drafting
- **Access:** Microsoft 365 subscription

---

## Developer-Focused Tools

### GitHub Copilot
- **Best for:** Code completion, boilerplate generation, in-editor AI
- **Strengths:** Tight IDE integration (VS Code, JetBrains), understands your codebase
- **Access:** GitHub subscription

### Cursor
- **Best for:** Full-codebase AI editing and chat
- **Strengths:** Can read and edit multiple files, context-aware code generation
- **Access:** [cursor.sh](https://cursor.sh)

### Claude Code (Anthropic)
- **Best for:** Agentic coding tasks from the terminal
- **Strengths:** Can run commands, read files, and execute multi-step coding workflows
- **Access:** CLI tool via Anthropic

## Choosing the Right Tool

```
Need to write or analyze a document?      → Claude or ChatGPT
Need to code?                             → GitHub Copilot or Cursor
Working in Google Workspace?              → Gemini
Working in Microsoft 365?                 → Microsoft Copilot
Need an image?                            → DALL·E or Midjourney
Need research with citations?             → Perplexity AI
Need to transcribe a meeting?             → Otter.ai
```

## Evaluating New AI Tools

Before adopting a new AI tool, ask:

1. **Data privacy** — Where does my data go? Is it used for training?
2. **Security** — Is it compliant with your organization's standards?
3. **Accuracy** — Does it cite sources? How do you verify outputs?
4. **Cost** — Free tier limitations? Enterprise pricing?
5. **Integration** — Does it fit into existing workflows?

See [06-data-privacy.md](./06-data-privacy.md) for more on evaluating tools safely.
