# 03 — AI Use Cases by Role

AI tools are versatile. This guide maps common job functions to practical, ready-to-use AI applications.

---

## 💼 Managers & Executives

| Use Case | Example Prompt |
|---|---|
| Summarize long reports | "Summarize this 20-page report. Focus on risks, decisions needed, and key metrics." |
| Draft communications | "Write a company-wide update about our Q2 results. Tone: transparent, optimistic." |
| Prepare for meetings | "Given these agenda items, generate 5 discussion questions to guide the team." |
| Performance feedback | "Help me write constructive feedback for an employee who is technically strong but struggles with communication." |
| Strategic brainstorming | "What are the pros and cons of expanding into the Eastern European market in 2025?" |

---

## 👩‍💻 Developers & Engineers

| Use Case | Example Prompt |
|---|---|
| Code generation | "Write a Python function that reads a CSV and returns rows where the 'status' column equals 'active'." |
| Debugging | "Here's my error message and code. What's causing this and how do I fix it?" |
| Code review | "Review this function for readability, edge cases, and performance." |
| Documentation | "Generate a docstring for this function following Google style." |
| Learning new tech | "Explain how WebSockets work and give me a simple Node.js example." |

See [08-code-assistance.md](./08-code-assistance.md) for a full developer guide.

---

## ✍️ Writers & Content Creators

| Use Case | Example Prompt |
|---|---|
| First drafts | "Write a 500-word blog post about remote work trends in 2025. Tone: conversational." |
| Editing | "Improve the clarity and flow of this paragraph without changing the meaning." |
| SEO optimization | "Suggest 10 SEO-friendly H2 headings for an article about project management tools." |
| Social media | "Write 5 LinkedIn post variations for this product announcement. Keep each under 150 words." |
| Repurposing content | "Turn this blog post into a Twitter thread with 8 tweets." |

---

## 📊 Analysts & Data Teams

| Use Case | Example Prompt |
|---|---|
| Data interpretation | "Here are our churn numbers by month. What patterns do you see?" |
| SQL assistance | "Write a SQL query to find the top 10 customers by revenue in the last 90 days." |
| Report writing | "Turn these statistics into a 2-paragraph executive summary." |
| Hypothesis generation | "Based on this user behavior data, what are 3 possible explanations for the drop in engagement?" |
| Dashboard copy | "Write clear, non-technical labels and tooltips for these KPI cards." |

---

## 🛎️ Customer Support & Success

| Use Case | Example Prompt |
|---|---|
| Response templates | "Write a polite, empathetic response to a customer complaining about a billing error." |
| FAQ generation | "Given these support tickets, generate a list of the 10 most common questions and concise answers." |
| Tone adjustment | "Rewrite this support response to sound warmer and less robotic." |
| Escalation summaries | "Summarize this long email thread so an escalation manager can get up to speed quickly." |

---

## 🎓 Learners & Researchers

| Use Case | Example Prompt |
|---|---|
| Concept explanation | "Explain transformer architecture to someone who knows basic Python but no deep learning." |
| Study guides | "Create a 10-question quiz on the French Revolution at a high-school level." |
| Literature review | "Here are 5 paper abstracts. What are the common themes and gaps in the research?" |
| Argument structuring | "Help me outline an essay arguing for universal basic income. Include counterarguments." |
